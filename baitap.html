git là gi? các lệnh cơ bản và công dụng của chúng? • Git là một hệ thống điều khiển phiên bản (version control system) theo hình thức phân tán • Git được sử dụng để quản lý mã nguồn (source code) và ghi nhận các thay đổi • GitHub là một dịch vụ Git được cung cấp miễn phí • GitHub có phiên bản trả phí dành cho các doanh nghiệp -các câu lenh của git và công dụng • git clone: Sao chép một repository • git init: Khởi tạo một repository • git add: Đưa các file vào trong vùng staged • git commit: Ghi nhận các thay đổi • git push: Đưa các thay đổi từ local repository lên remote repository

HTML Là gì? • HTML là ngôn ngữ được sử dụng để tạo ra các Webpage • HTML là viết tắt của chữ Hypertext Markup Language (Ngôn ngữ Đánh dấu Siêu Văn bản) • Đánh dấu: sử dụng các thẻ • Siêu văn bản: có thể đi đến văn bản khác thông qua các liên kết (link)
